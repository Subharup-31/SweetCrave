### Data Loading Strategy

1. **Initial Load**
   - Show loading spinner
   - Fetch products from Firestore
   - Render products
   - Hide loading spinner

2. **Real-time Updates**
   - Set up Firestore listeners
   - Auto-update on changes
   - No page refresh needed

3. **Cart Management**
   - Store in localStorage
   - Sync across pages
   - Persist on refresh
